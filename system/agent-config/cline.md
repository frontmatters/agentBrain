# Cline Agent Config

Cline can read a rule file from `~/Documents/Cline/Rules/agentBrain.md`.

## Cline-specific rules

- Follow `system/agent-config/shared.md` and `system/rules.md`.
- `scripts/setup.sh` creates a thin `agentBrain.md` pointer when Cline/VS Code extension paths are detected.
- Keep the Cline rule file as pointers to canonical agentBrain files.
