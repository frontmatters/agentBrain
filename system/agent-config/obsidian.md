# Obsidian Config

Obsidian is not an agent runtime; it is the human knowledge-browser for agentBrain.

## Obsidian-specific rules

- Open the agentBrain checkout as a vault.
- Use graph view, backlinks, and search for navigation/review.
- Obsidian plugin state stays untracked; only minimal vault config is public.
- Private notes remain under `local/` and sync through the private local repo.
