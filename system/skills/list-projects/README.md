# list-projects

Full catalog of `local/projects/` regardless of status.

## Purpose

Broader view than list-parks: shows every project with status, priority, date, and one-line description. Useful for audit, finding stale or status-less projects, or just "what projects do I have?".

## Usage

```
/list-projects
/list-projects --status active
```
