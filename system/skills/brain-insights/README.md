# brain-insights

Private reflection skill for recent Pi sessions and agentBrain notes.

## Purpose

Surface work patterns, repeated friction, missing learnings, and high-leverage improvements from recent sessions.

## Usage

```bash
/brain-insights
/brain-insights 30 days
/brain-insights agentBrain
```

## Output

By default the skill writes a private report to:

```text
local/reports/brain-insights-YYYYMMDD.md
```

The report should summarize evidence and recommended actions without copying secrets, raw tokens, private endpoints, or customer data.
