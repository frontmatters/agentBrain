# list-learnings

Read-only catalog of captured learnings.

## Purpose

Sibling of save-learning. Lists `local/learnings/` entries by recency with date, tags, and title — useful for end-of-project review, pattern-spotting across sessions, or finding a learning you remember writing but can't place.

## Usage

```
/list-learnings
/list-learnings --tag <tag>
```
