# save-troubleshoot

Log a problem and solution in agentBrain.

## Purpose

Records a bug fix, workaround, or cross-platform solution to `local/learnings/troubleshooting.md` or a project-specific note.

## Usage

```
/save-troubleshoot
```

Entries must include: problem, cause, solution, and context (when this occurs).
