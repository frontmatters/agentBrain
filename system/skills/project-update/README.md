# project-update

Create or update a project folder in `local/projects/`.

## Purpose

Creates a new project subfolder with the required `index.md` and optional files, or updates an existing one with new milestones, decisions, or status changes.

## Usage

```
/project-update
```

See `templates/project-*.md` for file templates.
