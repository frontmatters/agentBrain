# opensrc

Fetch dependency source code for deeper implementation context.

## Purpose

When an agent needs to understand how a library works internally, this skill fetches the source code from npm, PyPI, or crates.io and makes it available for analysis.

## Triggers

- "fetch source for X"
- "how does X work internally"
- "read the source of X"
- "get the implementation of X"

See `system/integrations/opensrc.md` for the full integration spec.
