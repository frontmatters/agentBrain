# brain-retro

Private retrospective skill for agentBrain itself.

## Purpose

Review the brain as a system: architecture, extensions, add-ons, backlog, and knowledge hygiene.

## Usage

```bash
/brain-retro
/brain-retro monthly
/brain-retro addons
/brain-retro extensions
```

## Output

By default the skill writes a private review to:

```text
local/reviews/brain-retro-YYYYMMDD.md
```

The report should summarize strengths, weaknesses, and the next highest-leverage improvements.
