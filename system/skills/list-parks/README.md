# list-parks

Dashboard of parked (paused or blocked) projects.

## Purpose

Read-only sibling of park / unpark. Filters `local/projects/` to status `paused` or `blocked`, sorted by last-touched. Best opened at session start to answer "what was I working on?".

## Usage

```
/list-parks
```

## Related

- [[park]] — pause a project with handover prompt
- [[unpark]] — resume a parked project in a fresh session
- [[list-projects]] — broader unfiltered catalog
