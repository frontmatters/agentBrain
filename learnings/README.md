# Learnings

Public `learnings/` is for **structure and examples only**.

## Rule

- Public = **HOW** and **WHERE** learning is stored.
- Private = **WHAT** was learned.

Do not commit real discoveries, tool evaluations, troubleshooting history, project-specific lessons, YouTube notes, or personal preferences here.

Real learnings belong in:

```text
local/learnings/
local/research/
local/memories/
local/projects/<project>/
```

Promote to public only when explicitly asked to publish a generic, sanitized example.
